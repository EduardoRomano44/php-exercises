# php-exercises
Exercises to learn PHP, using XAMPP.

## Author
- Eduardo Romano

## How to run

### Windows

Run a PHP file with PowerShell:

```powershell
./scripts/run.ps1 <file.php> [php-path]
```
Default path is ```C:\xampp\php\php.exe```

### Bash / WSL / Git Bash

Run a PHP file with Bash:

```bash
./scripts/run.sh ./[file.php] [php-path]
```

## Requirements

- XAMPP installed with PHP available, or PHP available in your terminal PATH
